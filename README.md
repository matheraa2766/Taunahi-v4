## 🌾 Farming Automation

### Pest Control System

| Feature | Description |
|---|---|
| **Pest Cleaner** | Fully automatic pest cleaning with realistic spraying mechanics |
| **Sprayonator / Repellent Integration** | Automatic tool usage triggered after pest cleanup |
| **Vacuum Bag Management** | Auto bag-emptying with Phillip for extra Farming Fortune |
| **Seamless Transitions** | Instant switching between Pest and Visitor scripts |
| **Fractional Navigation** | Advanced pathfinding through special blocks (cocoa, trapdoors) |

### Farm Builders

| Design | Description |
|---|---|
| **Compact Pumpkin & Melon** | High BPS profit design optimized for S-Shape scripts |
| **Vertical Wart & Crops** | Combined farm supporting all three crops and netherwart |
| **Classical Mushroom** | Traditional S-Shape mushroom farm layout |
| **Staircase Mushroom (SDS)** | Optimized staircase design for maximum XP gain |
| **S-Shape Wart/Crop** | Quick temporary farm solution for rapid deployment |

- **Daedalus Swap** — Automatic weapon swap for magic find optimization on killing blows

## 🎣 Fishing Automation

### Fishing Systems

| System | Description |
|---|---|
| **Worm Fishing (Solo)** | High-compatibility worm fishing with enhanced feature set |
| **Worm Fishing (Party)** | Coordinated party fishing synced with other Taunahi users |
| **Trophy Fishing** | Specialized script for Golden Fish and rare catches (Slugs) |
| **Hyperion Killing** | Integrated Hyperion usage for efficient lava fishing |

### Fishing Utilities

- **Auto-Heal Integration** — Continuous health management during fishing sessions
- **Totem Placing** — Automatic Totem of Corruption placement
- **Double Casting Fix** — Resolves slow fishing / double cast issues
- **Auto Buy Rain** — Automatic rain refill when in the Park
- **Auto Buy Bait** — Bazaar integration for seamless bait replenishment
- **Pool Warping** — Smart AOTV usage for efficient pool navigation and exit
- **Power Orb Placing** — Automatic stat buffing and mana regeneration orbs
- **Better Trash Removal** — Intelligent item disposal with configurable filters
- **Lava Avoidance** — Smart lava navigation without requiring Magma Lord armour
